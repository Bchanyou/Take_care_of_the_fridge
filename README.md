## 🤾‍♂️ 트러블슈팅
1.품목추가
해결방법 > 여러개의 값을 한번에 넣는게 아니라 배열로 받아서 값을 여러번 반복해서 넣을수 있도록 변경.  
 
2.캘린더
문제점 : 유통기한 달력 위 표시하면 품목이름이 떠야하는데 왼쪽 상단에 영어문자로 표기.
해결방법 > pom.xml 라이브러리와 이클립스 버전 달라서 1.7에서 11로 업그레이드 시킴.

3. 비밀번호 변경
문제점 : 리셋할때 메일로 보내는데 메일이 안옴.
해결방법 > 원래 계획은 gmail에서 발송하려고 했지만, 보안과 설정상의 문제가 발생해서
          보안 및 설정변경이 용이한 네이버 메일로 대체.
          pop3/smtp설정, imap/smtp설정 변경을 통해 포트 설정 후 네이버 메일에서 회원 비밀번호 찾기 메일을 보내도록 구현.

4. 레시피
문제점 : 유통기한 임박 재료들(3일전부터) 레시피 추천,연결.
         db에서 받아온 레시피 데이터를 다시 db로 보낼때 오류 발생. 
         서블렛에서 서블릿으로 통신할때 값이 안넘어가는 문제 발생.
   
해결방법 > 하나의 서블렛으로 묶고 맵퍼 설정을 바르게 했더니 수정 완료.

6. 고객센터
문제점 : 1:1 문의글 작성안됨, 게시판 작성, 문의사항 글작성 오류, 같은 제목의 글 작성시 view가 선택되지 않는 오류가 발생.

해결방법 > 제목과 글 작성 시간을 받아서 검색하도록 수정.
